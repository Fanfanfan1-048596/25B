# B题 项目

本目录包含第 B 题的代码和附件。该 README 提供基本说明和 GitHub 推送步骤。

忽略项（见 `.gitignore`）：
- `__pycache__/`
- `paper/`
- `思路.*`
- `问题一二.*`

快速开始：

1. 在本地初始化并提交（已提供示例）
2. 在 GitHub 上创建一个新仓库
3. 将远程添加为 `origin` 并推送：

   git remote add origin <your-github-repo-url>
   git branch -M main
   git push -u origin main

如果需要，我可以替你完成远程仓库的添加与推送（需要你的 GitHub 权限/Token）。
